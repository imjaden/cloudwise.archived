# cloudwise-honors

> 云智慧（2020-2026）在职期间个人贡献归档：荣誉表彰、团队赋能、解决方案输出与行业分享记录。

🔗 **在线访问**：[cloudwise.archived.jaden.tech](https://cloudwise.archived.jaden.tech)

## 项目说明

本项目记录了本人在云智慧（北京）科技有限公司任职解决方案架构师期间（2020年7月 — 2026年6月）的工作成果与荣誉，包括：

- **贡献与荣誉**：11 项时间线条目，涵盖公司级荣誉、客户感谢信、行业峰会、团队建设等
- **核心贡献概览**：解决方案输出、团队赋能库建设、内部培训与分享
- **华东区周一早分享**：2023-2026 年累计 13 次技术分享记录

## 技术栈

- 单文件 HTML（HTML + 内联 CSS + 内联 JS）
- 深色主题（`#0A0A2A` 深海蓝）
- 响应式设计，支持移动端
- Lightbox 图片放大查看
- 无外部依赖，零构建成本

## 部署

本项目通过 GitHub Pages 部署，绑定自定义域名 `cloudwise.archived.jaden.tech`。

```bash
# 本地预览
open index.html
```
